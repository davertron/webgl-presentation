What is OpenGL?

A standard specification defining a cross-language, cross-platform API for writing applications that produce 2D and 3D computer graphics.

Vertices & Primitives

Matrices

Used to represent transformations (translation, rotation, scaling)

Do I have to Do Math???

Not really. There's just two matrices you care about: ModelView and Projection, and you use libraries to help you manipulate the matrices.

ModelView Matrix

Basically the matrix that represents the position/scale/rotation of your "model" (collection of vertices)

Projection Matrix

Defines a clipping volume (clipping just means "don't draw vertices outside of this volume")

Textures

Just an image that you map to a set of vertices
